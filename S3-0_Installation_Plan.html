EMC Installation Plan — Seven Steps
Section 3 — Installation & Filters
1
List All Components and Define Zones
Inventory all EMI sources (drives, switchgear) and victims (PLCs, sensors, radio, GPS). Define separation zones: power, control, signal.
2
Design the Grounding System (Star + Backplate)
PE star point at the main switchboard. Unpainted backplate for drives. Each drive chassis bonds to backplate by direct metal contact — no paint, no gaskets.
3
Specify Cable Routes and Segregation
Power cables (AC mains, motor): ≥ 300 mm from signal cables. Never route motor and control cables in the same trunking. Minimum crossing angle 90° if cables must cross.
4
Select Motor Cable Type and Glands
4-conductor shielded cable with ≥ 85% braid coverage. Foil+braid preferred. 360° EMC cable glands at drive entry plate and motor terminal box. Shield grounded at both ends.
5
Select and Install Input Filter (if Required)
RFI filter on the mains input side of the drive. Most drives have internal filters — verify the category is correct for the installation environment. External filter for upgraded attenuation.
6
Select Output Filter (du/dt, Sine-Wave, or CM)
Based on cable length, motor type, and bearing EDM risk. See Section 3 filter screens for selection rules. Not all installations need an output filter — overspecifying adds cost and losses.
7
Verify and Document
Measure leakage current, verify IT system compatibility (marine), confirm cable shield continuity, run the drive and check for interference on connected systems. Record installation as designed.
Plan Before You Wire
EMC problems discovered after installation are expensive to fix. Cable routing and grounding topology cannot be changed without rewiring. Plan them at the design stage — not during commissioning.
Key Checklist Item — Paint
The single most common cause of EMC installation failure is paint between the drive mounting flange and the backplate. Remove paint at every mounting contact surface. Seal the bare area with clear lacquer after installation to prevent corrosion.
Equipotential Conductor
Install a ≥ 10 mm² bonding conductor between the drive chassis, motor frame, and panel PE. This is required by EN 61800-3 and limits the voltage difference between equipment that shares a CM current path.
Key Concept
Good EMC results from design decisions made before the first cable is installed: zone separation, grounding topology, cable type, and gland selection. Retrofitting EMC fixes after the fact costs three to ten times more than designing correctly from the start.
