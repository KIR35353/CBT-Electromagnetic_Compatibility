CM Core Installation — Correct vs. Incorrect
Section 4 — Bearing Electric Stress
Cross-Section — Correct and Incorrect Core Installation
Correct Installation
Pass all three phase conductors (U, V, W) through the core in the same direction. The PE conductor, cable screen, and any control cables must pass outside the core. Multiple cores may be stacked — consult the product datasheet for minimum number and stacking order.
Critical Error — PE Inside Core
The PE conductor carries the CM return current in the opposite direction to the phase conductors. If PE passes through the core, it cancels the CM flux attenuation. The filter will appear installed but provide no reduction in bearing voltage. Confirm PE routing in your installation checklist.
Critical Error — Missing Phase
If only two phases pass through, the three-phase current sum is no longer zero for differential-mode current. The core saturates — its permeability collapses, it generates heat, and in severe cases the core can overheat and damage adjacent wiring. This error also voids the product warranty.
Verification After Installation
After fitting: measure shaft-to-ground voltage with an oscilloscope before returning to service. Target: Vshaft < 10 V peak. Also measure core surface temperature during first 30 minutes of operation — it should not exceed 60 °C. A hotter core indicates a wiring error.
Key Concept
All three phase conductors must pass through the core — PE must not. If PE passes through the core the filter is defeated. If a phase is missing the core saturates and overheats. Verify with shaft voltage measurement and core temperature check after installation.
