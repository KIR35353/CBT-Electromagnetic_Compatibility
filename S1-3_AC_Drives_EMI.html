Why AC Drives Generate EMI
Section 1 — What Is EMC?
Common-Mode Current Path in an AC Drive System
The du/dt Source
The inverter switches DC bus voltage at frequencies of 2–16 kHz, producing output voltage transitions at 1–10 kV per microsecond. This high rate of voltage change is unavoidable in modern IGBTs.
Parasitic Capacitance
Between the motor winding and the motor frame, between the cable conductors and the cable shield — these are real capacitances of 10–100 nF. Combined with the high du/dt, they produce common-mode current: I = C × dV/dt.
The Two Return Paths
CM current must return to the drive. If a shielded cable is correctly bonded at both ends, it returns through the cable shield — contained and harmless. Without this path, it returns through the mains supply, causing interference on all connected equipment.
Y-Capacitors Provide a Local Path
Inside the drive, Y-capacitors in the RFI filter and DC link provide a local return path from the motor cable's CM current back to the DC bus. This is why the RFI filter must be intact and correctly installed — and why Y-cap bypass switches matter in IT distribution systems.
Key Concept
AC drives are inherently common-mode noise sources: ICM = Cpar × du/dt. The installation designer's job is to provide a low-impedance return path for that current — through the cable shield — so it does not flow through the mains supply, the vessel hull, or other sensitive circuits.
